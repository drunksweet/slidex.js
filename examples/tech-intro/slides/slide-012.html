Layer A · 页面级
tang.onLoad()
每次翻到此页时触发，驱动整页元素的入场序列。 支持可视化 禁用/启用 或切换框架预设效果。
// 整页脚本 <script> tang . onLoad (() => { gsap.from( '.title' , { y: - 20 , opacity: 0 , duration: 0.5 }) gsap.from( '.card' , { y: 30 , opacity: 0 , stagger: 0.08 }) })
Layer B · 元素级
data-step-1="N"
属性声明式绑定，演示时第 N 次点击才显示。 支持多绑定：同一元素可绑定多个步骤（入场 + 出场）。
<!-- HTML 属性或可视化配置 --> <div data-step-1 = "1" data-animation-1 = "fade-up" data-duration-1 = "450" data-ease-1 = "power2.out" > 第 1 次点击后显示 </div>
Layer C · 自由级
tang.onStep(N, fn)
脚本注册任意回调，在第 N 步触发完全自定义动画逻辑。 点击"步骤 4"可见下方进度条实时驱动演示 →
// 自由控制任意 DOM tang . onStep ( 4 , (els) => { gsap.to( '#prog-fill' , { width: '100%' , duration: 1.2 , ease: 'power2.out' }) })
