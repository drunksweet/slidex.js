Layer A · 页面级入场动画
tang.onLoad( ) — 整页入场
每次翻到此页时触发，驱动整页元素的入场 GSAP 序列。 编辑模式可一键 禁用 / 启用，或切换为框架预设效果。
// slide.html <script> tang.onLoad(() => { gsap.from('.title', { y: -20, opacity: 0, duration: 0.45 }) gsap.from('.card', { y: 30, opacity: 0, stagger: 0.08 }) })
Layer B · 元素级步骤动画
data-step="N" — 点击逐步展现
为元素添加 data-step="N" 属性，演示时第 N 次点击才显示。 支持 10 种预设效果 + 时长/延迟/缓动全量配置。
<!-- 在右侧面板可视化配置，或直接写属性 --> <div data-step="1" data-animation="fade-up" data-duration="450" data-ease="power2.out"> 内容在第 1 次点击后显示 </div>
10 种内置预设效果
✨淡入 ⬆️向上淡入 ⬇️向下淡入 ➡️从左淡入 ⬅️从右淡入 🔍缩小出现 🔎放大出现 🚀从下滑入 💫从左滑入 ⚡无效果
